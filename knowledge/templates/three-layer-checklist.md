# 内容三层结构优化检查清单
- **用途**: 逐项检查页面是否完成宏/中/微三层优化
- **来源**: 论文6(Section 4.4)
- **预期效果**: 三层协同引用率 +17.3%

## 宏观结构（贡献44.9%）— 最重要

- [ ] H1标签存在且包含【品牌名】+【产品类别】
- [ ] 每个主要板块有H2标题（至少4个H2）
- [ ] H2下无跳级（H2→H3，不跳H4）
- [ ] H2标题语义清晰、可独立理解
- [ ] 页面顶部有目录或锚点导航

## 中观结构（贡献39.7%）

- [ ] 至少1个对比表格（你 vs 竞品）
- [ ] 至少1个FAQ折叠面板（3-5个问答）
- [ ] 至少1个列表（ul/ol，非纯段落）
- [ ] 至少1个数值事实区块（stats）
- [ ] 每个信息块独立、可被单独引用

## 微观结构（贡献15.4%）

- [ ] 品牌名每次出现时加粗（**BrandName**）
- [ ] 关键数值加粗（**4.8/5** rating）
- [ ] 产品特色用视觉标记突出（如 ✅ 图标）
- [ ] 段落短小（3-4句以内）
- [ ] 每段第一句包含该段核心信息

## 可提取证据检查（每页至少2种）

- [ ] 定义句："X is [品类] designed for [场景]..."
- [ ] 数值事实："50% lighter than market average"
- [ ] 比较："vs Competitor X, we offer..."
- [ ] 步骤："How to: Step 1 → Step 2 → Step 3"

## AI引擎差异化检查

- [ ] ChatGPT偏好: FAQ + 对比表 + 明确结论 ✅
- [ ] Gemini偏好: YouTube链接 + 视觉内容 + 新鲜度 ✅（如适用）
- [ ] Perplexity偏好: 深度内容 + 多源引用 ✅
- [ ] Claude偏好: 数据支撑 + 逻辑严密 + 引用可靠 ✅
