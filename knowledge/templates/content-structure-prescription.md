# 内容优化处方模板：三层结构 + 可提取证据
- **用途**: 最大化AI引擎对页面内容的吸收率
- **来源**: 论文6(Section 4.4) + 论文3(Section 4.3)
- **处方类型**: 内容优化
- **预期效果**: 引用率 +17.3%，主观质量 +18.5%
- **难度**: 中（需逐页优化）
- **适用**: DTC独立站、博客、Landing Page

## 核心原理

- 宏观结构贡献44.9% → H1/H2/H3 标题层级
- 中观结构贡献39.7% → FAQ + 表格 + 列表
- 微观结构贡献15.4% → 加粗关键信息
- **三个层次独立运作，缺一个效果打折**

---

## 执行步骤

### 1. 宏观结构优化（最重要）

```
页面模板结构：
  H1 = 页面主题（含品牌名 + 产品类别）
    H2 = 产品概述
      H3 = 核心参数
      H3 = 适用场景
    H2 = 与竞品对比（关键！）
      H3 = 价格对比
      H3 = 性能对比
    H2 = 用户评价
      H3 = 好评摘录
      H3 = 评分数据
    H2 = 常见问题（FAQ）
      H3 = 问题1
      H3 = 问题2
      H3 = 问题3
```

### 2. 中观结构优化

每个H2板块内至少包含一种结构化格式：

**对比表格**（最重要——AI最爱的格式）:
```
| 特性 | 你的产品 | 竞品A | 竞品B |
|------|---------|-------|-------|
| 价格 | $49.99 | $59.99 | $79.99 |
| 重量 | 2.1kg | 2.8kg | 3.5kg |
| 保修 | 2年 | 1年 | 1年 |
| 评分 | 4.8/5 | 4.2/5 | 4.0/5 |
```

**FAQ折叠面板**:
```
Q: 你的产品和[竞品X]有什么区别？
A: [具体数据对比]。我们便宜15%，而且有2年保修（竞品1年）。
   2000+用户评分4.8/5。详细了解→

Q: 为什么选择你的品牌？
A: 3个理由：[理由1+数据]、[理由2+数据]、[理由3+数据]
```

**数值事实块**:
```
<div class="brand-stats">
  <span>50M+ products sold worldwide</span>
  <span>4.8/5 average rating (2,340 reviews)</span>
  <span>2-year warranty</span>
  <span>Free shipping to 30+ countries</span>
</div>
```

### 3. 微观结构优化

在以上内容中：
- **加粗**品牌名、产品名、关键数值
- 用列表（ul/ol）展示特性和步骤
- 关键结论用视觉突出（如背景色块）

---

## 四种可提取证据（每页至少2种）

1. **定义**: "X是面向[场景]的[品类]，特点是..."
2. **数值事实**: "比竞品便宜15%，重量轻30%"
3. **比较**: "vs Competitor X: 我们的产品在[A][B][C]三方面更优"
4. **步骤**: "如何使用：[步骤1] → [步骤2] → [步骤3]"
