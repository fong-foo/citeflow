# 权威建设处方模板：Earned Media + Cite Sources
- **用途**: 提升品牌在AI引擎中的权威信号
- **来源**: 论文1(Section 4) + 论文4 + 论文5(Section 3.4) + 论文17
- **处方类型**: 权威建设
- **预期效果**: 中长期效果，6-12个月建立可持续引用
- **难度**: 高（需持续运营）
- **适用**: 所有品牌，尤其中国跨境品牌

---

## 第一步：获取第三方权威引用（Earned Media）

### 目标清单（按优先级）

| 优先级 | 渠道 | 为什么重要 | 怎么做 |
|--------|------|-----------|--------|
| 🔴 最高 | 行业媒体评测 | AI最信任的信号 | 送样品给Wirecutter/Tom's Guide/行业垂直媒体 |
| 🔴 最高 | Wikipedia提及 | Google/Perplexity核心数据源 | 确保品牌在相关Wiki页面有客观提及 |
| 🟡 高 | YouTube评测 | Gemini引用偏好 | 联系海外科技/行业YouTuber做评测 |
| 🟡 高 | Reddit讨论 | Perplexity预测信号 | 在相关subreddit自然参与讨论 |
| 🟢 中 | 媒体PR | 品牌知名度 | 在TechCrunch/The Verge/行业媒体有报道 |

### 模板：联系媒体评测的邮件

```
Subject: [Product Name] for review — [1 key stat that makes it interesting]

Hi [Name],

I'm from [Brand], a [country]-based [category] brand.

Our [product] has one number worth noting: [最打动人的数据，如"50% lighter than the market leader"].

We've sold [数量] units across [国家数] countries with a [评分] rating.

I'd love to send you one for an honest review. No strings — keep it if you like it, or don't cover it if it's not a fit.

Best,
[Name]
[Brand]
```

## 第二步：在国际平台建立存在（针对中国品牌）

### 必须建立的国际平台存在

1. **Wikipedia**
   - 确保品牌在相关品类页面有提及（不需要独立页面）
   - 如果品牌足够知名，创建 Wikipedia 页面（需要独立、可靠的第三方来源）

2. **Reddit**
   - 在 r/BuyItForLife, r/[你的品类], r/GoodValue 等 subreddit 自然参与
   - **不要自推**——回答别人的问题，提到你的品牌时自然引用数据
   - 做 AMA（Ask Me Anything）："我是[品牌]的产品经理，做了X年的[品类]，AMA"

3. **Medium / Dev.to / 行业博客**
   - 发布深度对比文章："[你的产品] vs [竞品]：哪个更适合[X场景]？"
   - 包含真实数据、测试结果、用户反馈

4. **Product Hunt**
   - 发布产品并获得 upvotes
   - PH排名是Perplexity中的可见性预测信号（论文5）

5. **G2 / Trustpilot**
   - 鼓励用户在这些平台留评价
   - 聚合评价是AI的权威信号
