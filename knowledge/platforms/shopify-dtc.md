# Shopify DTC 独立站 GEO 策略
- **适用**: Shopify/WooCommerce/自建独立站
- **最后更新**: 2026-05-20

## DTC GEO 优势
- **完全控制**: robots.txt / Schema / 页面结构 / 内容格式 → 所有GEO杠杆都可用
- **品牌资产积累**: 独立站优化积累的效果属于你自己（不像Amazon会被平台规则重置）
- **全栈优化**: 技术+内容+权威+社区四类处方全部可执行

## 核心策略

### 1. 技术地基（一次性设置）
- robots.txt 放行 GPTBot + Google-Extended + anthropic-ai + PerplexityBot
- Product Schema（每个产品页）
- FAQPage Schema（关键页面）
- BreadcrumbList Schema（帮助AI理解网站结构）
- 页面加载 < 3秒

### 2. 产品页三层结构（每个SKU都要）
按照三层结构模型（论文6）：
- H1 → H2×4+ → H3
- 对比表格 + FAQ面板 + 数值统计块
- 加粗品牌名和关键数值

### 3. 内容飞轮
- /blog 发布品类对比文章、购买指南、使用教程
- /compare 页面：你的产品 vs 竞品A vs 竞品B
- /glossary 品牌术语表
- 每月至少2篇新内容

### 4. 外部信号
- YouTube产品评测（Gemini引用源）
- Reddit社区存在（Perplexity引用源）
- Medium深度文章（ChatGPT引用源）
- 行业媒体评测（权威信号）
