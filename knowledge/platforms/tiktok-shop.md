# TikTok Shop GEO 策略
- **适用**: TikTok Shop 卖家（跨境/本土）
- **最后更新**: 2026-05-20

## TikTok Shop GEO 特殊性
- **内容即货架**: 视频内容 = 产品发现渠道 + AI训练数据源
- **不可控性高**: 你不能优化TikTok的技术层面
- **间接GEO**: TikTok Shop不是AI直接引用的目标（AI引用TikTok内容描述，不引用Shop页面）

## 核心策略

### 1. 视频内容 → AI训练数据
- TikTok视频的文字层（字幕、描述、标签）进入LLM训练数据
- 你的产品如果在热门视频中出现 → 品牌名和关联场景进入AI记忆
- 策略：鼓励创作者在视频描述中@你的品牌，使用品牌话题标签

### 2. 创作者生态
- 合作的创作者在YouTube同步发布长视频 → 双重覆盖（TikTok+YouTube）
- YouTube版本做详细评测 → Gemini引用来源
- 创作者在Reddit/其他平台提及你的品牌 → 社区信号

### 3. 品牌一致性
- TikTok、独立站、Amazon上的产品信息必须完全一致
- AI跨平台引用时不一致会降权（论文3暗示）

### 4. TikTok Shop 的 GEO 局限
- **不能直接优化**: 你无法在Shop页面加Schema或改结构
- **推荐做法**: TikTok Shop做销量和曝光 → 独立站做GEO优化 → 形成循环
- TikTok带来发现 → 用户搜索品牌名 → AI给出引用独立站的回答 → 用户在独立站完成购买
