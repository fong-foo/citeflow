Screenshot Checklist

  • All screenshots at 2x resolution (2560px wide) for retina displays
  • Dark theme: #0A0A0F background, no browser chrome / OS window decorations
  • Cursor hidden — no hover states visible
  • Use real brand data with non-trivial results (not demo/empty states)
  • File format: PNG, optimized (pngquant or similar, target <500KB each)
  • Naming: exactly as specified above, placed in img/ directory
